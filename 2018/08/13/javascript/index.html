Skill

    trampoline 蹦床函数（把递归函数转化为循环的函数，防止调用栈过多造成 maximum call exceed 错误）

    比较字符顺序, sort()默认使用字符的asicii码比较, 会导致不是按a-z顺序, 使用 localeCompare比较顺序

    把字符串中匹配模式的的字符串组成数组并返回 相当于 regexp的exec方法

  • includes 方法

    判断给定字符串是否包含字符

  • substring 方法

    截取字符串

  • startsWith, endWith

    判断字符串开头, 结尾
